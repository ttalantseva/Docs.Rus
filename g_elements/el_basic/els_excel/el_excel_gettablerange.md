# Получение диапазона таблицы

Предназначен для получения диапазона таблицы Excel.

| Свойство         | Тип             | Описание                                     | Пример
| ---------------- | --------------- | -------------------------------------------- | ---------------------------------------- |
| Диапазон         | String          | Диапазон таблицы. Обязательно для заполнения | "A1:D12"
| Таблица          | String          | Наименование таблицы. Обязательно для заполнения | "Сотрудники"
| Страница         | String          | Наименование страницы                        | "Данные"
| Индекс страницы  | Int32           | Номер страницы, отсчет с 0                   | 0
